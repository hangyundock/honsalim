## 2. 시나리오 분류 매트릭스

### 2-1. 3축 분류

```
              [페르소나]                  [예산 구간]              [시즌]
        ┌──────────────────┐        ┌──────────────────┐    ┌──────────────────┐
        │ A. 새내기 자취생  │        │ S 5~20만원       │    │ 2~3월  신학기 1차 │
        │ B. 재택근무자    │   ×    │ M 20~50만원      │ ×  │ 8~9월  신학기 2차 │
        │ C. 1인 정착자    │        │ L 50~100만원     │    │ 11~1월 홈오피스   │
        │                  │        │ XL 100만원+      │    │ 1월   새해미니멀  │
        └──────────────────┘        └──────────────────┘    │ 봄가을 이사철     │
                                                            └──────────────────┘

3 × 4 × 5 = 60 슬롯. 초기 10편은 우선순위 높은 슬롯만 채움. 잔여는 확장 큐.
```

### 차별화 의무 (M3 [확정 세션 #6 Google AI Optimization Guide 정합])

60 슬롯 모두 **진짜 다른 가치** 의무. 단순 변형 (예: "30만 자취" vs "35만 자취"만 다른 글) **금지**:
- ❌ commodity content: "7 Tips for First-Time Homebuyers" (Google 예시)
- ✅ non-commodity content: 페르소나 × 예산 × 시즌 결합 고유 인사이트
- 각 시나리오는 다른 페르소나·시즌·예산 구간 결합 → 검색 의도·소비 패턴 차별화 의무
- 동일 페르소나 내 예산 구간 (S/M/L/XL) 도 단순 가격 차이 X — 구간별 우선순위·전략 차별화 의무 (예: S 절약 우선·M 균형·L 만족도·XL 프리미엄)

위반 시 Google "확장된 콘텐츠 악용 = 스팸 정책 위반" 패널티 [확정 Google].

10편 모두 동일한 카드 구조 따름. 본문 생성·검증·디자인의 기준.

### 5-1. 글 1편 구조 (8섹션)

```
[Hero] 시즌·페르소나·예산 한 줄 + 첫머리 disclosure (E1)
   │
[1] 누구를 위한 가이드인가 (페르소나 짧은 정의)
[2] 왜 이 예산·시즌인가 (시즌 검색 의도 정당화)
[3] 추천 상품 6~12개 (각 200~300자 + 직접 사진 or 공식 위젯)
[4] 예산 분배표 (각 카테고리 배분)
[5] 함께 보면 좋은 시나리오 2~3편 링크 (페르소나 매칭)
[6] FAQ 3~5개 (검색 의도 보강)
[7] 푸터 disclosure + 사업자 정보 (E3, 등록 후) + Schema JSON-LD
[8] 클릭 추적 footer 메모 ("본 글의 모든 상품 링크는 /go/ 자체 게이트웨이를 거칩니다")
```

### 5-2. 진실성 강화 요소 (E7·E8 회피)

- **사용자 직접 사진 1~3장 의무**: hero 또는 추천 상품 중 1~3개에 사용자 본인이 보유·촬영한 사진 첨부 (D5 [확정])
- **1인칭 한국어 허용** (E8): "써본 결과", "내 원룸에서는" — 단 직접 사진 첨부된 상품 한정
- **추정 표현 명시**: "[관찰]" "[추정]" 본문에도 등급 사용 가능 (선택)
- **가격 변동 안내**: "가격은 글 작성 시점 기준입니다" 문구 자동 삽입 (BACKEND)

### 5-3. SEO 요소 (F1~F5 [확정] 적용)

- 제목 50~70자 (검색 결과 잘림 방지)
- meta description 150자
- BreadcrumbList Schema (Home → 페르소나 → 시나리오)
- ItemList Schema (추천 상품 6~12개)
- Article Schema (본문 메타)
- Review Schema는 **직접 사진 + 사용자 1인칭** 둘 다 있는 상품만 (F5 [확정])
