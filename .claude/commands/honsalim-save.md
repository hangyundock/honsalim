# /honsalim-save — 혼살림 중간 저장

세션 도중 중간 저장. 세션 종료 없음. EVENTS.md 미작성.

작업 순서:

1. `D:\affiliate_hub\docs\STATE.md` 일부 갱신 (변경 영역만)
2. `D:\affiliate_hub\docs\TODO.md` 활성 작업 갱신 (완료 제거·신규 추가)
3. `D:\affiliate_hub\docs\DECISIONS.md` 새 [확정] 사실 있으면 카테고리 A~H 분류 후 추가
4. **EVENTS.md 작성 금지** (세션 종료 시만)
5. **git commit 금지** (세션 종료 시만)
6. 사용자에게 갱신 결과 요약 보고 (diff 줄 수 + 변경 핵심)

용도: 장시간 세션 중간 백업, 사용자 휴식 전, 작업 분기점.

규칙:
- 변경 후 read 검증
- 비개발자 친화 — diff 요약만 채팅 출력
- 사용자 확인 후 적용 (변경 전 승인)
