# faq_generate.md — FAQ 3~5개 생성 보조 프롬프트

> 출처: SCENARIOS §5-1 + FRONTEND §4-2 [확정]
> 사용: 본문 생성 시 자동 포함되거나, FAQ 누락 시 별도 호출.

다음 세팅에 대한 FAQ 3~5개를 작성하세요.

세팅: {{scenario.title_ko}}
라이프스타일: {{persona.title_ko}}
예산: {{scenario.budget_min_krw}}~{{scenario.budget_max_krw}}원

## 좋은 FAQ 기준

- 사용자가 실제 검색할 만한 자연스러운 질문
- 한국어 구어체 ("...해요?" "...할까요?")
- 답변은 본문 데이터 기반 (창작 정보 금지)
- 1개 질문당 답변 50~150자
- 의료·법무·금융 단정 회피

## 회피 패턴 (POLICY §3-1-4)

- "이 제품은 정말 훌륭한가요?" 같은 자기 칭찬 유도 질문 금지
- "혼살림은 어떤 사이트인가요?" 같은 사이트 광고 질문 금지

## 출력 형식

```json
[
  {"q": "질문 1", "a": "답변 1"},
  {"q": "질문 2", "a": "답변 2"},
  {"q": "질문 3", "a": "답변 3"}
]
```

JSON 외 다른 텍스트 출력 금지.
